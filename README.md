# Dark Matter Rotation Curve Analysis
Analyzes rotation curve data from the SPARC database to estimate dark matter 
fractions in NGC 3198, NGC 6503, and DDO 154.

## How to run
1. Install requirements: pip install numpy pandas matplotlib scipy
2. Download SPARC data files into the /data folder
3. Run: python dark_matter_analysis.py
